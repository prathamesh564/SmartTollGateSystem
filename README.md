# Smart Toll Gate System using ESP32

## Overview

The Smart Toll Gate System is an IoT-based automated toll collection solution developed using ESP32. The system identifies authorized vehicles using RFID technology and automatically opens the gate upon successful verification. It reduces manual intervention, minimizes traffic congestion, and enables efficient toll management.

## Features

* Automated toll collection using RFID cards/tags
* ESP32-based wireless controller
* Servo motor-controlled toll gate
* Real-time vehicle authentication
* LCD/OLED display for status messages
* Fast and secure access control
* Reduced waiting time at toll plazas

## Components Used

* ESP32 Development Board
* RFID Reader (RC522)
* RFID Tags/Cards
* Servo Motor
* LCD/OLED Display
* Jumper Wires
* Breadboard
* Power Supply

## Working Principle

1. Vehicle approaches the toll gate.
2. RFID tag is scanned by the RC522 reader.
3. ESP32 verifies the tag information.
4. If the tag is valid, the servo motor opens the gate.
5. Vehicle passes through the gate.
6. Gate automatically closes after a predefined delay.
7. Transaction details can be logged for monitoring and analysis.
